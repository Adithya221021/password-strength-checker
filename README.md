# Password Strength Checker

## Overview

The Password Strength Checker is a cybersecurity-focused Python application designed to evaluate password security using multiple threat intelligence techniques. It analyzes password complexity, detects weak patterns, and checks exposure against known compromised password databases.

This tool demonstrates secure coding practices, threat analysis, and defensive security implementation aligned with real-world cybersecurity standards.

---

## Cybersecurity Features

• Password complexity analysis  
• Detection of weak and common passwords  
• Breach detection using HaveIBeenPwned API  
• Secure hash verification using SHA-1  
• Real-time strength feedback  
• User-friendly interface  

---

## Security Techniques Used

• SHA-1 hashing for breach comparison  
• K-Anonymity model for privacy protection  
• API security implementation  
• Input validation and sanitization  
• Secure coding practices  

---


## Installation

Clone the repository:

```bash
git clone https://github.com/Adithya221021/password-strength-checker.git
cd password-strength-checker
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the tool:

```bash
python password_checker.py
```
